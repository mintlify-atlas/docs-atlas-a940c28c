## Why Faker?

Faker provides a single, consistent API to generate realistic test data across every layer of your application stack — from unit tests to end-to-end browser automation.

## Get started in seconds

<Steps>
  <Step title="Install the package">
    <CodeGroup>
      ```bash npm
      npm install --save-dev @faker-js/faker
      ```
      ```bash pnpm
      pnpm add --save-dev @faker-js/faker
      ```
      ```bash yarn
      yarn add --dev @faker-js/faker
      ```
    </CodeGroup>
  </Step>
  <Step title="Import and generate data">
    ```typescript
    import { faker } from '@faker-js/faker';

    const user = {
      id: faker.string.uuid(),
      name: faker.person.fullName(),
      email: faker.internet.email(),
      avatar: faker.image.avatar(),
      birthdate: faker.date.birthdate(),
      registeredAt: faker.date.past(),
    };
    ```
  </Step>
  <Step title="Use locale-specific data (optional)">
    ```typescript
    import { fakerDE as faker } from '@faker-js/faker';

    faker.person.firstName(); // 'Lena'
    faker.location.city();    // 'München'
    ```
  </Step>
</Steps>
