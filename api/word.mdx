title: "faker.word — adjectives, nouns, verbs, and more"
description: "Generate individual words by part of speech: adjectives, adverbs, conjunctions, interjections, nouns, prepositions, verbs, and samples with faker.word."
---

`faker.word` returns individual English words organized by part of speech. Every method accepts an optional length constraint and a strategy that controls what happens when no word in the dictionary matches the requested length.

## Length and strategy options

All single-word methods share the same options signature:

<ParamField body="options" type="number | object">
  Pass a number as shorthand for `{ length: number }`, or pass an options object.
</ParamField>

<ParamField body="options.length" type="number | { min: number; max: number }">
  The exact character length of the word, or a min/max range.
</ParamField>

<ParamField body="options.strategy" type="string" default="'fail'">
  How to handle cases where no word matches the requested length.

  - `'fail'` — throws an error if no match is found
  - `'closest'` — returns the word whose length is nearest to the target
  - `'shortest'` — returns the shortest word available
  - `'longest'` — returns the longest word available
  - `'any-length'` — ignores the length constraint and returns any word
</ParamField>

<Tip>
The default strategy for `faker.word` methods is `'fail'`, which differs from `faker.lorem.word()` where the default is `'any-length'`. Set `strategy: 'closest'` or `strategy: 'any-length'` when you want a best-effort result rather than an error.
</Tip>

## adjective()

Returns a random adjective.

```typescript
faker.word.adjective()                                            // 'pungent'
faker.word.adjective(5)                                           // 'slimy'
faker.word.adjective({ strategy: 'shortest' })                    // 'icy'
faker.word.adjective({ length: { min: 5, max: 7 }, strategy: 'fail' }) // 'distant'
```

## sample()

Returns a random word from any part of speech: adjective, adverb, conjunction, interjection, noun, preposition, or verb.

```typescript
faker.word.sample()   // 'incidentally'
faker.word.sample(5)  // 'fruit'
```

### Parameters

Same length and strategy options as the single-part-of-speech methods above.

<Note>
`sample()` tries each word list in a random order. If a length constraint is used and only some parts of speech have a matching word, it silently skips the ones that don't and returns the first successful result. An error is only thrown if every word list fails.
</Note>

## words()

Returns multiple words from any part of speech, joined by spaces. Defaults to 1–3 words.

```typescript
faker.word.words()                              // 'almost'
faker.word.words(5)                             // 'before hourly patiently dribble equal'
faker.word.words({ count: 5 })                  // 'whoever edible um kissingly faraway'
faker.word.words({ count: { min: 5, max: 10 } }) // 'vice buoyant through apropos poised total wary boohoo'
```

### Parameters

<ParamField body="options" type="number | object">
  Pass a number as shorthand for `{ count: number }`, or pass an options object.
</ParamField>

<ParamField body="options.count" type="number | { min: number; max: number }" default="{ min: 1, max: 3 }">
  The number of words to return, or a min/max range.
</ParamField>
