---
title: "faker.animal — animal types and species names"
description: "Generate animal names by type including bear, bird, cat, cetacean, cow, crocodilia, dog, fish, horse, insect, lion, rabbit, rodent, snake, and more."
---

The `faker.animal` module generates animal names organized by taxonomic or common grouping. Use `type()` to get a generic animal category, or call a specific method such as `cat()` or `dog()` to get a breed or species name. All values may be localized.

## type()

Returns a random general animal type.

```typescript
faker.animal.type()
// 'crocodile'
```

## petName()

Returns a random pet name.

```typescript
faker.animal.petName()
// 'Coco'
```

## bear()

Returns a random bear species.

```typescript
faker.animal.bear()
// 'Asian black bear'
```

## bird()

## Usage example

```typescript
import { faker } from '@faker-js/faker';

const animals = [
  { type: faker.animal.type(), name: faker.animal.petName() },
  { type: 'dog', name: faker.animal.dog() },
  { type: 'cat', name: faker.animal.cat() },
  { type: 'bird', name: faker.animal.bird() },
];

// [
//   { type: 'crocodile', name: 'Coco' },
//   { type: 'dog', name: 'Irish Water Spaniel' },
//   { type: 'cat', name: 'Singapura' },
//   { type: 'bird', name: "Buller's Shearwater" }
// ]
```
