The `faker.datatype` module provides generation of fundamental data types. It is locale-independent and has no external dependencies on locale definitions.

## Methods

### boolean

Returns a random boolean (`true` or `false`). You can bias the result toward `true` by providing a probability greater than `0.5`.

<ParamField path="options.probability" type="number">
  The probability of returning `true`, between `0.0` and `1.0`. The value is limited to two decimal places.
  Defaults to `0.5`.
  - Values `<= 0.0` always return `false`.
  - Values `>= 1.0` always return `true`.
</ParamField>

<CodeGroup>

```javascript basic usage
faker.datatype.boolean()
// false

faker.datatype.boolean(0.9)
// true

faker.datatype.boolean(0.1)
// false
```

```javascript with options object
faker.datatype.boolean({ probability: 0.75 })
// true   (returned 75% of the time)

faker.datatype.boolean({ probability: 0.3 })
// false  (returned 70% of the time)
```

</CodeGroup>

<Note>
  A `probability` of `0.75` means `true` is returned approximately 75% of calls. The probability is rounded to two decimal places internally.
</Note>
